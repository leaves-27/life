# 区块链的定义：
  区块链是给很多东西提供基础的信任网络的一种去中心化技术系统。

# 比特币
  比特币是一种数字货币，其使用了区块链技术系统。

# 比特币的实现
比特币网络是一个个本地保存的完整数据库互联链接
个人钱包是一个个密钥锁定的未消费交易输出
区块链是一个个包含很多个交易的区块串联起来的链
