# 以太坊概述
  用来实现区块链应用的平台,使开发者可以专注于业务上方便的开发基于区块链的应用。

# Truffle概述
  Truffle是针对基于以太坊的Solidity语言的一套开发框架。本身基于Javascrip.

# solidity
  Solidity是一种语法类似JavaScript的高级语言。它被设计成以编译的方式生成以太坊虚拟机代码.
