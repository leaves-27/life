# 动作:
一只脚在板上，一只脚反复进行蹬地、悬空

# 技巧：
保持重心一直在前脚。保持的方法：
  
  蹬地时，蹬地脚侧肩膀微后倾，板上脚进行屈膝；蹬地结束后蹬地脚侧肩膀先前倾后恢复，板上脚站直。

  蹬地时机是在失去太多速度前，每次蹬地不需要太用力。

  抬腿的时候不要后仰

第一阶段: 轻松、流畅的进行滑行；
