# 羽毛球技术动作
  (1)挑球: 低点位前场球挑到后场.
  (2)扑球: 在球到达网顶时，跳起将球向下扑下。
  (3)推球: 高点位球由前场推倒后场
  (4)搓球: 发球线和网之间的网前球的处理,斜切球托
