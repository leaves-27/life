# 概述
在自然界中，人类性象与动物有很大不同（男性睾丸比大猩猩大比黑猩猩小，阴茎比他们都大；女性隐形排卵；私下进行性活动）。其原因作者认为是由于人类的食物来源广泛且人类获取这种多元的食物来源的方法复杂，需要群体内成员的共同参与，进而导致人类的性象演化为如今的模样。


人类对自身为什么会演化出当前的繁殖系统(性器官、性活动)还知之甚少。作者试图根据已知道的信息试图给出自己的答案（即他认为是为了）。
一、当前的繁殖系统：
大的方面狩猎采集社会轻微的一夫多妻制、现代社会一夫一妻制；更细微的方面：
1.性器官：男人身高比女人高、男人有浓密的体毛和胡须、男人阴茎在动物界中最大但没有什么实用价值、女性的乳房也没有找到实用价值、男人睾丸比黑猩猩小比大猩猩大。
2.性活动：两性进行私下性交且频繁性交却大多数性交不能产生后代、女性进行隐性排卵


二、知之甚少的原因在于：
1、问题本身困难（不管是从解剖学还是生理学有些问题目前都解释不了）。
2、人类对性活动的态度，使得问题难以进行有效研究。（之前不重视，现在无法以实验的方法进行客观研究）
3、牵扯太广（涉及到社会系统与食物来源相关）。

三、他认为的原因：
有些特征是为了实际需要的适合、有些是为了吸引异性、有些是为了排挤竞争者、有些是为了促进合作保证下一代的生存、成长。

对于隐性排卵和私下性交为什么会演化出来，作者认为是由于人类的食物种类繁杂且取得食物依赖的技术、工具复杂以及对抗其他生存风险，使的父母双方及更多社会成员之间需要相互长期合作，才能维持自身和幼儿的生存、成长。即人类的食物和其他生存挑战塑造了隐性排卵和私下性交这种生命周期特征。

食物：种类繁杂（分散）、获取要使用的技术复杂。
社会系统：社会系统由异性同性群居组成，基本单位是两性组成的家庭（即合作是主流）。

人类掌握觅食需要的要素：
肢体结构上，肢体得发育到一定程度的灵活性。
知识和经验上，大脑得习得食物相关的知识和掌握相关的经验。


四、其他知识点：
人类形态结构特征的演化可以通过研究历史遗留的化石证据来获得，但人类生命周期特征的演化只能依赖和人类的近亲（猿类）比较获得。因为人类生命周期特征没有留下什么可供研究的化石证据。

亲代与子代的关系（即亲代抚养不抚养子代）、两性之间的关系（即是保持一对一的交配关系还是一对多或多对多的交配关系）
寿命：

人天生并不适合高效的采集和打猎，但从学习如何利用、加工工具以及如何进行合作中，走向了食物链的顶端。
