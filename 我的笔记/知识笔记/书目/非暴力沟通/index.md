# 非暴力沟通的应用场景：
  1.爱护自己
  2.促进家庭和睦
  3.改善工作交流
  4.发展友谊
  5.推动政治和解  

# 这本书可以传授的东西:
  避免自己无心或有意的语言暴力，使人与人之间情意想通、和谐相处。具体又可以分为
  1.诚实的表达自己(自己的需要)
  2.关切的倾听他人(他人的需要)

## 无心的语言暴力
言语上的指责、否定、嘲讽、说教以及任意打断、拒不回应、随意出口的评价和结论。
