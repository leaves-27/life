# 概述
保留日常工作可以致富，但需要在业余时间去创办一家公司来学习成为伟大的企业家相关的技能才行。

# 成为企业家的首要原则是
不要单纯地为钱工作，而应该是工作是为了学到长远的技能。

## 销售工作的好处
1.可以帮助客服恐惧心理
2.推销可以帮助发展获取财富所需的技能
3.保留工作，在业余时间创业。
4.伟大的企业家不等同于可以制造出卓越产品的人。即要想成为伟大的企业家，就不要绞尽脑汁的去生产最好的产品，而应该集中精力去创办一家企业，成为如何成为卓越的企业家。
