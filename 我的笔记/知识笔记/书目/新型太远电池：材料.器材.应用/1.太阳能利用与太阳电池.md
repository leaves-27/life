# 概述
1.太阳能的重要性：地球上的大部分能源(风能、水能、海洋温差能、波浪能和生物能、潮汐能以及化石能源)本质上都是来源于太阳能。其每4分钟投射到地球表面上的能量足够全球人全年的能量消耗;每秒钟投射到地球表面的能量相当于500万吨煤燃烧的能量。
2.太阳能的来源：太阳能是由于太阳内部连续不断的核聚变反应产生的能量。然后经由对流过程，以光和热的形式传递到地球表面。
3.太阳能与可再生能源的关系：可在生能源是指原材料可以再生的能源。由于太阳的年龄到目前为止大约为46亿年，还可以再燃烧50亿年。而人类到目前为止的历史才五千多年，所以相对于人类的历史来说，太阳能就是可再生能源。

# 太阳能与可再生能源
1.风能: 
  (1)风能的来源：由于地面受太阳辐射气温变化不同和空气中水蒸气含量不同，引起的各地气压的差异，导致高压空气向低压空气流动产生的。
  (2)风能的缺点：受地形影响较大，一般产生于沿海和开阔大陆的收缩地带。
  (3)风能的作用分类：主要分为风力发电和作为动力两种。
2.生物能:
  (1)生物能的来源：是太阳能以光化学反应的形式存储在生物体内的一种能量。它直接或间接的来源于光合作用(即二氧化碳和水在太阳能的作用下合成有机物并释放出氧气)，其对光的转换效率为百分之几，具体作用机理尚不太清楚。

3.地热能:
  (1)地热能的来源：来自地球内部的熔岩，以热力形式存在，以地下水的流动或熔岩的形式涌至地面。
  (2)利用地热能的形式：按其温度不同来分。
    <1>200~400摄氏度：直接发电或综合利用
    <2>150~200摄氏度：双循环发电，制冷，工业干燥，工业热加工
    <3>100~150摄氏度：双循环发电，供暖，制冷，工业干燥，脱水加工，回收盐类，罐头食品。
    <4>50~100摄氏度：供暖、温室、家庭用热水、工业干燥.
    <5>20~50摄氏度：水产养殖、饲养牲畜、土壤加温、脱水加工。
4.水能:
  (1)水能的来源：来源于水的动能、势能和压力能等能量资源。包括河流水能、潮汐水能、波浪能、海流能等。
  (2)水能的利用：水力发电还处于岂不阶段，目前主要采用河流、潮汐、波浪、涌浪等水的运动来发电。

# 太阳能光伏应用
1.原则上，太阳能可以转换成任何形式的能量。目前应用主要转换为热能、电能(转换为电能主要说的就是太阳能电池)。 
2.太阳能利用涉及的技术问题很多，根据其技术特点，主要分为以下4项：
  (1)太阳能采集
  (2)太阳能转换
  (3)太阳能储存
  (4)太阳能传输

# 太阳电池的发展
1.1954年~1960年：单晶硅太阳电池出现，其转换效率为6%。硅材料的植被工艺日趋完善，硅材料质量不断提高，其转换效率提高到15%。
2.1972年~1985年：背电场电池技术、”浅结“结构、绒面技术、密栅金属化等技术的应用使电池效率提高到17%，成本大幅下降。
3.1985年后，各种电池新技术和新材料的出现改进了电池性能，提高了光电转换下来。

## 全球发展趋势
1.自1954年第一块实用的硅太阳电池被在实验室制成后，其就被应用到了各种太空中的飞行器中，但由于当时的发电成本过高，在地面中并不流行。
2.其后在1970年左右，由于能源短缺、生态破坏和环境污染，我们为了寻找到一条可持续发展的清洁能源道路，所以又将目光集中到了太阳能上面。各国相继出台各种各样的政策法规开始发展太阳能产业降低发电成本、提高转换效率。
3.1999年开始，光伏产业快速发展。
4.2010年开始到目前，光伏组件产量增长近10倍，价格下降了四分之三。

### 电池效率
1.1839年法国人在化学反应中发现了光伏效应；1876年固态se也发现了光伏效应。
2.1887年第一块太阳电池问世，其转换效率为1%到2%。
3.1941年硅太阳电池出现。
4.1954年单晶硅太阳电池问世。
5.1958年硅太阳电池首先在航天器上得到应用。
6.1970年，许多新技术引入电池制造工艺，转换效率得到了很大提高。到1980年左右时候，硅太阳电池已经开始大量引入地面应用。

## 我国发展太阳能的必要性
1.随着我国经济的发展，能源紧缺已成为我国经济持续发展的瓶颈。
2.我国太阳能极为丰富。全国三分之二的土地平均日照2000小时以上。
3.2013年至今，我国相继出台了一系列政策发展光伏产业。
