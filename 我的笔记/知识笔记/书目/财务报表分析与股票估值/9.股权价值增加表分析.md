# 基于资产资本表的利润表重构：股权价值增加表
## 标准利润表的缺陷
1.标准利润表无法与标准资产负债表或资产资本表之间建立起清晰的逻辑联系。即无法从标准利润表看出那些收入、哪些利润是哪些资产产生的。
2.标准利润表只考虑扣减了债务资本成本，而没有考虑股权资金成本，导致决策者无法根据标准利润表判断和分析一家公司是否真正创造了价值。
3.标准利润表自身内部混乱。现行标准利润表会误导人们认为利润总额与所得税费用之间存在对应关系，从而忽略和不考其他综合收益与所得税费用之间的关系。
## 利润表的重构: 股权价值增加表
1.利润表重构后的股权价值增加表构成：
  (1)经营利润: 经营利润 = 经营收入 - 经营成本 - 销售费用 - 管理费用 - 营业税及附加 - 资产减值损失 + 营业外收入 - 营业外支出
  (2)股权投资收益：对联营企业和合营企业的投资收益
  (3)金融资产收益：金融资产收益 = 投资收益 - 对联营企业和合营企业的投资收益 + 公允价值变动净收益 + 汇兑收益 + 其他综合收益

  (4)股权资金成本
  (5)债务资本成本

2.股权价值增加表中的概念解释：
  (1)息税前经营利润: 息税前经营利润 = 经营收入 - 经营成本 - 销售费用 - 管理费用 - 营业税及附加 - 资产减值损失 + 营业外收入 - 营业外支出
  (2)息前税后经营利润: 息税前经营利润 = 经营收入 - 经营成本 - 销售费用 - 管理费用 - 营业税及附加 - 资产减值损失 + 营业外收入 - 营业外支出 - 息税前经营利润 * 利润所得税税率
  (3)息前税后金融资产收益: 息前税后金融资产收益 = 投资收益 - 对联营企业和合营企业的投资收益 + 公允价值变动净收益 + 汇兑收益 + 其他综合收益 * (投资收益 - 对联营企业和合营企业的投资收益 + 公允价值变动净收益 + 汇兑收益 + 其他综合收益) * 金融资产税率 
  (4)税前利润(息后税前利润): 息税前经营利润 + 金融资产收益税前利润 - 财务费用
  (5)真实财务费用: 
  (6)净利润: 息税前经营利润 + 金融资产收益税前利润 - 财务费用 - 所得税
  (7)价值增加值: 价值增加值 = 净利润 - 股权资本成本 * 0.08
# 股权价值增加表分析：营业收入分析
## 营业收入历史增长情况
  从一家公司多个年度的营业收入增长情况，可以判断该公司的成长性。常见的成长类型为：
  1.成长型公司: 营业收入逐年快速增长。
  2.衰退型公司: 营业收入逐年下降。
  3.成熟型公司: 营业收入各年间保持稳定。

## 营业收入未来增长空间判断
  由于营业收入是由多项不同业务的收入加总后得出的，因此，对其未来增长空间进行分析时，应该根据不同业务进行详细分析。决定某项业务营业收入未来增长空间的因素是：是否有扩大销量的可能和是否可以提高单价。
  1.是否有扩大销量的可能：要扩大销量需要从行业本身的增长空间和企业的增长空间两方面来考虑。若公司已经是头部的垄断寡头，市场占有率已经非常高，那么行业本身增长空间没有增长的话，其未来扩大销量的空间就很小。
  2.是否可以提高单价：是否可以提高单价取决于公司是否有定价权。没有定价权提高单价，势必会导致销量下滑。
## 营业收入分析的特殊问题：收入的确认问题
  收入的确认是指收入入账的时间和金额的确认。因其确认需要会计人员根据自主判断空间进行确认，所以，存在很多利润操作的可能。此处常见的利润操作有两种：
  1.在不同年度之间进行跨期调节和粉饰。其不会导致某一长段时间区间总收入的变化，但会影响某一小段时间收入的变化趋势，给使用者带来误导。
  2.完全虚构收入：即没有真实业务发生，完全进行收入造假。这个可以结合现金流分析进行规避。因为虚构收入无法带来真实的现金流。

  收入的确认分类：
  1.时间：即收入是在什么时候计入账册的。在售前、售中还是售后。
  2.金额：即是以什么金额登记的。是总额法还是净额法，是完工百分比还是合同百分比。
# 股权价值增加表分析：营业成本分析(营业成本必须低于营业收入)
## 成本性态分析：亏损的业务是否就应该停产
亏损的业务是否停产取决于业务的边际贡献。边际贡献为负的业务停产，边际贡献为正的业务继续。业务的边际贡献又取决于销售价格与变动成本的差。变动成本定义如下：
在企业管理中，成本按形态划分为：固定成本费用、变动成本费用、半变动或半固定成本费用。
1.固定成本费用：不受业务量增减变化影响而保持不变的成本。其是沉没成本，不管做不做都会发生。
2.变动成本费用：随业务量的变动而成正比例变动。其是相关成本，不做就不会发生。

单位产品成本 = 固定成本分摊 + 单位变动成本。 
## 经营资产结构决定营业成本结构
即重资产行业的公司，一般固定成本就高、变动成本就低；而轻资产行业，一般固定成本较低、变动成本就高。
产能过剩的原因是由行业竞争格局和成本结构决定。充分竞争导致个体的理性引起群体的非理性(即行业已经过剩的情况下，个体都不愿意自己是被淘汰的那一个，而争相扩大生成规模来降低平均成本，进而导致整个行业不断过剩)。
## 营业成本控制能力的分析
由于营业成本是由多项不同业务的成本加总后得出的，因此，对其营业成本控制能力进行分析时，应该根据不同业务进行详细分析。对于单项业务，由于营业成本等于单位成本乘以销量，而单位成本又等于单位固定成本加上单位变动成本。因此，营业成本控制能力分析，需要从单位固定成本和单位变动成本两个角度来分析。
1.从控制单位固定成本的角度来考虑：我们需要对公司业务规模做一个大致的预测和判断，在固定成本总额不变的情况下，业务规模越大，则单位固定成本越低。
2.从控制单位变动成本的角度来考虑：需要考虑公司内部的精细化管理程度。例如：采购管理、精益生产方式。
## 营业成本分析的特殊问题：成本与收入匹配原则
成本与收入匹配原则，即一家公司某会计期间或某个会计对象所取得的的营业收入应与为取得改收入所发生的营业成本相匹配，以便可以正确计算该会计期间的营业利润。一些公司为了操作利润，在某些年度少结转营业成本，导致虚增公司利润的同时虚增公司存货。这时候，我们需要结合公司的毛利率以及存货周转率等指标，来进行分析和判断公司是否存在着财报舞弊问题。
# 营业收入和营业成本的综合分析：毛利和毛利率
## 毛利分析
1.毛利的计算：
  毛利 = 营业收入 - 营业成本。而营业成本参照如下概念：
    销售费用：市场营销费用、广告费用、差旅费等。
    管理费用：研发费用、行政管理人员工资等。
    营业成本：不包括上述费用，仅包括生成商品和提供服务的直接人工成本、原材料以及相关的水电煤等成本。
2.毛利的作用：反映一家公司产品和服务的获利程度。其又由价格、成本和销量三者决定，这三者又会互相影响。例如：提高价格，可能会影响销量，使销量降低；而降低成本又可能影响质量，进而影响价格和销量。
3.毛利与营业收入的关系：毛利应该逐年与营业收入保持同幅度增长或者比营业收入增长幅度大。否则，即意味着营业成本增幅超过营业收入增幅，对公司增收不利；或者营业收入比营业成本下滑更快，公司陷入衰退境地。

1.毛利润率分析的角度：
  (1)毛利润率的高低。
  (2)毛利润率的稳定性。
2.毛利润率的高低和稳定性由公司所处的行业、公司竞争策略、公司管理效率决定。
  (1)公司所处的行业:
  (2)公司竞争策略:
  (3)公司效率：
3.在分析毛利率时，我们应该考虑下面几方面:
  (1)毛利率与公司战略一致吗？例如：行业门槛高，竞争者少，毛利率就高;行业门槛低，竞争者多，毛利率就低;采用差异化战略的公司通常比采用低成本战略的公司毛利率高。
  (2)毛利率变化了吗？变好了还是变坏了？底层原因是什么？
  (3)公司对制作成本和费用管理的怎么样？这些成本是由哪些经营活动产生？这些经营活动都是有必要的吗？
4.毛利率高低对行业的影响：
  (1)毛利率长期低于15%：该行业属于过度竞争是行业。
  (2)毛利率长期在15%~25%：该行业属于高度竞争的行业。
  (3)毛利率长期在25%以上：该行业竞争格局比较好。
  (4)毛利率波动性比较大：该行业受外部因素如宏观经济形势、上下游供应链等因素影响较大。
5.行业对毛利率的影响：
  (1)行业逐级衰退或消失时，行业直接决定一家公司的生死。
  (2)行业没有衰退，竞争者众多时，一般表明市场空间巨大，这时，公司的竞争策略和管理效率对毛利率的影响更大。
# 股权价值增加表分析：营业费用分析
## 营业费用的范围
  1.我们这里所说的营业费用的范围是营业税金及附加、销售费用、管理费用、资产减值损失、营业外收入、营业外支出。其与标准利润表存在不同。标准利润表的营业费用是营业营业税金及附加、销售费用、管理费用和财务费用。我们这里这样归类的原因是：
  (1)财务费用与筹资活动的关系更加密切，而与公司经营活动关系不大。因此，财务费用我们将其放到资金成本的分析逻辑中去。
  (2)虽然资产减值损失包括经营资产减值损失、长期股权投资减值损失和金融资产减值损失。但其是公司在经营活动中形成的(即一般是由于公司在使用过程中操作不当或最初购置决策失误引起)，且对于经营管理比较好的公司，一般数值不大,所以归入营业费用。
  (3)对于营业外收入和营业外支出，因为其一般是固定资产处置、无形资产处置、政府对公司经营活动的补助等，因此我们将其归入营业费用，但若其是由长期股权和金融资产引起的，应归到它们各自的收益。对于其金额巨大的情况，应进行详细分析归类。同样，对于经营管理比较好的公司，不合理的营业支出一般比较少，不必要。
## 营业费用分析的重点
  营业费用分析的重点在于将其和各个费用在总收入中的占比跟其在历史年份中作比较，看其是否稳定或有所下降。因为会赚钱的前提是要学会花钱。
  1.营业税金及附加率：反映一家公司在营运环节所承担的税费负担,一般应该与国家税法规定的税率相当。但由于税法对不同行业的营业税费及附加规定不同，因此，应该具体行业具体对待。
  2.销售费用率：其对于多数优秀公司而言，会有比较稳定的销售政策和销售渠道。因此，其一般应该比较稳定或略有下降。若显著上升，则需要进一步分析是销售销量下降还是市场竞争所致。一般市场竞争激烈导致的，则意味着整个行业格局转变或恶化。
  3.管理费用:反映一家公司在总部管理方面的绩效。一般好的公司管理费用率应比较稳定。如果呈上升趋势，需要具体分析到底是哪些原因引起的，并考虑是坏事还是好事。这里需要注意的是因为管理费用包含研究费用，有些公司不区分研究费用和开发费用，将其全部列到研究费用里面，导致管理费用虚高而专利技术等无形资产低估。而研究费用加大是好事。
  4.经营资产减值损失率：体现该公司在营运活动中的管理能力。其在计算时，应对股权资产减值损失和金融资产减值损失进行扣除。对于管理比较优秀的公司，其一般比较低且很稳定。若其在某几个年份大幅度上升，可能是公司为了特殊目的进行操纵利润，大比例计提资产减值损失导致。
  5.营业外收支率：其反映一家公司对营业外业务和事项的能力。因为其一般是由于公司经营活动导致的，其一般都非常低。如果过高，则要详细分析是否合理以及是否会持续发生。
# 股权价值增加表分析：利润分析
## 息税前经营利润、息前税后经营利润和营业收入利润率
  息税前经营利润和息前税后经营利润反映一家公司通过营业活动获取收益的绝对金额;而营业收入利润率反映一家公司通过经营活动获取收益的相对能力。对于一家发展有前景的公司，多个季度或年度息前税后经营利润应该稳步上升，营业收入利润率保持稳定或者略有上升。否则，发展前景堪忧。

  息税前经营利润 = 营业收入 - 营业成本 - 营业费用
  息前税后经营利润 = 息税前经营利润 * (1-企业实际所得税税率)
  营业收入利润率 = 息前税后经营利润 / 营业收入 * 100%
## 长期股权投资收益分析
  长期股权投资收益分析的重点在于其看持续性和增长性。这个主要从分析其收益总额及其趋势发展变化来看。因为对于投资者而言，若其投资了很多被投资单位的话，分析每一家被投资单位的财报一是成本太高，二是很多信息无法获得。当然最好是能够充分收集、整理被投资单位的各种信息进行详细分析，尤其是涉及的投资金额巨大的时候。

## 金融资产收益分析
  因为金融资产收益并非大多数公司的主要创造价值手段，而是辅助手段。其在资金充裕的时候进行金融资产投资，在经营活动需要资金的时候卖出金融资产。因此，其收益波动性一般比较大，在分析时需要关注其稳定性以及占息税前利润总额的比例。

## 从息税前利润结构看利润的质量
  1.金融资产收益: 对于非银行保险等金融类公司，其在利润中的占比小，持续性和稳定性差(即经常大起大落)，收益质量不高；但对于银行保险等金融公司，其是它们的主要收益，其在利润中的占比应该很高,持续性和稳定性应该很好。
  2.长期股权投资收益: 对于非投资型公司，其在利润中的占比小，持续性和稳定性较金融资产收益好，较经营利润差。但对于投资型公司，其是它们的主要收益，其在利润中的占比应该很高,持续性和稳定性应该很好。
  3.经营利润: 其是一般工商企业的主要利润来源，稳定性和持续性要比金融资产和长期股权收益要高。

# 不要仅仅为银行打工：财务费用和债务资本成本分析
  债务资本成本率是债务资本成本占资本的比率，其值越低，代表进行筹资活动的的成本率越低；而财务成本负担率是债务资本成本占息税前利润的比率，其值越低越好，代表利润中用来支付利息支出的比例越低，否则，越高则越表明公司在为银行打工。

  1.债务资本成本(即债务资本成本总额)范围：利息支出总额(由于标准资产表上的利息支出是利息支出与利息收入的差的净额，因此计算利息支出总额时，应该使用净额+利息收入)、外币债务汇兑损益(即汇兑损益中由于筹资外币债务资金导致的部分)、债务筹资手续费(即与债务资金筹集有关的手续费，不包含发行股票所支付的手续费、经营活动相关的手续费)、其他债务筹资费用(与债务资金筹集有关的其他费用)。
  2.债务资本成本率的计算：
    债务资本成本率 = 债务资本成本总额 / 债务资本平均金额 * 100%
    债务资本平均金额 = (期初债务资本 + 期末债务资本) / 2
  3.财务成本负担率 = 真实财务费用 / 息税前利润 = (息税前利润 - 税前利润) / 息税前利润 = 1- 税前利润 / 息税前利润
    税前利润 = 息税前利润 - 真实财务费用
    真实财务费用 = 标准利润表中的财务费用 + 利息收入
# 实际企业所得税负担分析
  实际企业所得税负担率代表息后税前利润中用来支付企业所得税的比例。其值越低，代表息后税前利润中用来支付企业所得税的部分越少，否则，则企业为税务局打工。
  实际企业所得税负担 = 1 - 净利润 / 税前利润 * 100%.
  净利润 = 税前利润 - 企业所得税费用
# 净利润、股权资金成本和股权价值增加值分析
## 净利润与股权价值增加值
  净利润在扣除股权资金成本后的部分即为股权价值增加值。若其为负数，则表明公司没有给股东创造价值，而是在损毁股东的价值；只有在其等于零或大于零的时候，才表明其是在给股东创造价值或给股东创造超额价值。

  股权价值增加值 = 净利润 - 股权资本 * 股东预期回报
## 利润和股权价值增加值的局限性
   股权价值增加值很高并不能表明一个公司具有广阔的发展远景。原因是：一、股价价值增加值是通过净利润调整而来的，而净利润可以通过利润操作而来(当然这一点可以通过拉长分析时间窗口周期来识别)；二、利润和现金流量之间具有差异性，即利润是账面上的盈利，而现金流量才是实实在在的拿在手里的盈利。
   
## 利润可持续性的考量
即使股权价值增加值很高、当前的现金流量也很高，也并不能反映一个公司具有广阔的发展远景。因为短期的股权价值增加值和现金流量都可以通过除利润操作以外的经营管理方法来美化和粉饰。而为了美化和粉饰当前的利润和现金流，减少对未来发展有关的投入会使得公司在未来竞争中处于劣势，甚至出局(即会导致未来利润和现金流断崖式下降)。所以，要看一个公司具有广阔的发展远景需要将股权价值增加值、当前现金流量和自由现金流量(考虑所有净现值为正的投资计划或增长机会所需资金后所剩余现金流量)结合起来看，才能反映一个公司的真正盈利能力。
