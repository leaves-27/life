# 概述

## 建立一个论证
逻辑推理的过程即是建立论证的过程。一个论证由两部分组成，前提和结论（前提和结论通过逻辑指示词来连接）。在这里有两点要注意：
1.复杂的论证往往有大量相互依赖的前提。即前提之间有顺序。
2.有效的论证往往得出一个简单的结论。即不要得出多个结论。

### 常用的逻辑指示词

## 从全称到特称
  从全称到特称得出的结论是必然的。

## 从特征到全称
  从特称到全称的结论是可能的。但这偏偏又是人类的天性，所以我们要加以谨慎对待。
