# 概述
在评估一只股票时，采用将其当做自己的企业进行评估的思维方式，可以使得你可以靠着自己日渐积累的个人经验形成的成熟判断来投资股票。
评估自己企业的思维方式需要你立足资产负债表，然后通过考虑由资产负债带来的盈利来评估企业的内在价值。
