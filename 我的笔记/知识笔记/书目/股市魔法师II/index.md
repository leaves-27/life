# 时刻跟着计划走

所有努力和成功开始前都必须有一个计划，其可以提供一系列具体的行动指南。而股票投资是一件很很严肃的事情，更需要有周密的行动计划。如果没有计划，你将在犹豫不决与遗憾后悔两种情绪之间摆动和斗争，产生内在的情绪冲突。
