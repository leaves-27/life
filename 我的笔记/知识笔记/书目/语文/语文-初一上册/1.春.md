# 初春的山、水和太阳
山：朗润起来
水：涨起来了
太阳：脸红起来了

# 春草
纵向：从土里钻出来，绿绿的、嫩嫩的
横向：园子里、田野里，瞧去，一大片一大片满是的
感觉：风轻轻的、草软绵绵的---触觉

# 春花
桃花：红的像火---视觉
杏花：粉的像霞
梨花：白的像雪

蜜蜂：嗡嗡的闹着--听觉
蝴蝶：飞来飞去

野花：遍地都是，散在草丛里，像眼睛，像星星,眨呀眨

# 春风
1.不再寒冷。不寒杨柳风。---触觉
2.夹带着泥土、青草和花香的味道。----嗅觉
3.应和着清脆的鸟鸣、牧童嘹亮的短笛。---听觉

# 春雨: 雨的特点、雨中景致
雨：像牛毛，像花针，像细丝，密密地斜织着。屋顶上笼着一层薄烟。树叶绿的发亮，小草青的逼眼
夜晚：一点点黄晕的光烘托出安静而和平的夜
人：撑伞走路的行人、地里工作的农夫

# 人:
天上的风筝
出来活动筋骨的人

# 总写春天的特点
新的、有旺盛生命力的：刚落地的娃娃
美的、活泼生动的：像小姑娘，花枝招展
健壮有力的：像健壮的青年，有铁一般的胳膊和腰脚
