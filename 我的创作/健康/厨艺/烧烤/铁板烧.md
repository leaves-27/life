# 概况
  起源于西班牙，发展壮大于日本。

# 日式----亲橙里大鱼(简约)
  选料上乘，一般选用最上乘的材料(不过差别也很大，牛肉有国产牛肉<50元一斤>、美国牛肉<85元一斤>、神户牛肉<700元一斤>)，制作之前不经过腌制，用油偏向黄油，装盘偏向简约，只在烧烤过程中加入盐、胡椒两种调味品，品的是食物的原始味道。

# 法式-----高端西餐厅(浪漫)
  用油偏向椰子油，偏向于各种蘸汁、偏向于精美的复杂的装盘。

# 韩式----卿古炭火烤肉(平实)
  韩国铁板烧主要以牛肉为主，以烤牛里脊和烤牛排最有名。讲究选料，选料非常严格，通常牛肉来自专门的肉联公司。
